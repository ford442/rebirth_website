# ReBirth RB-338 — Version 1.5

**Release date:** October 1997  
**Developer:** Propellerhead Software AB, Stockholm

## Overview

Version 1.5 was a substantial feature update released approximately seven months after
the initial launch. Community feedback directly shaped many of the additions in this
release, most notably the introduction of the **ReBirth Mod (.rbm)** file format that
allowed third parties to replace the device graphics and sample sets.

## New Features

### Effects Rack

Three new master effects were added to the signal chain:

| Effect        | Controls                                  |
|---------------|-------------------------------------------|
| Distortion    | Drive, Tone, Output Level                |
| Compressor    | Threshold, Ratio, Attack, Release        |
| Pattern Delay | Time (1/8 – 1/1), Feedback, Wet/Dry Mix |

### ReBirth Mod System (`.rbm`)

The `.rbm` format debuted in 1.5. A mod file is a ZIP-based container holding:
- Replacement bitmap graphics for any device panel
- Optional replacement PCM sample sets for the 808/909
- A `mod.ini` manifest declaring metadata (name, author, version)

Mods are applied at launch and are fully reversible by switching back to the default skin.

### Pattern Improvements

- **Copy / Paste** patterns across tracks
- **Randomize** button for instant generative acid patterns
- Expanded accent resolution (per-step accent + slide)

### MIDI Enhancements

- MIDI output for all devices — route 303/808/909 to external hardware
- MIDI CC mapping for knobs (per-device assignable)

## Bug Fixes

- Fixed occasional hanging MIDI note on fast pattern changes
- Corrected 303 filter self-oscillation amplitude at maximum resonance
- Addressed rare crash on Windows 95 OSR2 during file save
