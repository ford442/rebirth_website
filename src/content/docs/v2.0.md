# ReBirth RB-338 — Version 2.0

**Release date:** September 1998  
**Developer:** Propellerhead Software AB, Stockholm

## Overview

Version 2.0 was a ground-up revisit of the core audio engine, coinciding with growing
competition from early DAWs and hardware samplers. Propellerhead invested heavily in
filter accuracy and introduced BeatMap — a revolutionary feature for the era — allowing
ReBirth to synchronise to audio by analysing tempo from an audio file.

## Core Engine Changes

### Improved Filter Model (PCF)

The TB-303 filter model was completely rewritten using a new **Proportional-Coefficient
Filter (PCF)** algorithm. The new model more accurately reproduced:

- Transistor ladder saturation characteristics
- Resonance overshoot and feedback clipping behaviour
- Self-oscillation sine purity at Resonance = max

Community members performing A/B comparisons with hardware units widely considered the
2.0 filter a step-function improvement over the 1.x model.

### BeatMap

BeatMap allowed ReBirth to analyse a WAV or AIFF file and lock its internal sequencer
to the tempo grid of that recording. This enabled:

- Overdubbing ReBirth onto live drum recordings
- Synchronising with DJ mixes or field recordings
- Tempo-correct bounce to tape or DAW

### Pattern Enhancements

- Song mode extended from 16 to **64 patterns** per device
- Triplet grid option for 303 patterns (12-step per beat subdivision)
- Full copy-paste of entire device states between patterns

## MIDI Overhaul

- Latency compensation configurable per soundcard
- SysEx dump and restore of all device states
- External MIDI clock jitter filtering

## Mod System Updates

- `.rbm` v2 manifest format with richer metadata (artwork, description, web URL)
- Support for 256-colour and 16-bit colour panel graphics
- Mod manager UI for browsing, activating, and deactivating installed mods

## Platform Support

| Platform      | Notes                          |
|---------------|--------------------------------|
| Windows 95/98 | Fully supported                |
| Windows NT 4  | Supported (with ASIO driver)   |
| Mac OS 8.x    | PowerPC only from this release |
| Mac OS 9      | Supported from 2.0.1 patch     |

## Known Issues

- BeatMap analysis may produce incorrect tempo on recordings with heavy swing
- Mod Manager occasionally loses mod order on application restart (fixed in 2.0.1)
