# ReBirth RB-338 — Version 1.0

**Release date:** March 1997  
**Developer:** Propellerhead Software AB, Stockholm

## Overview

Version 1.0 marked the debut of ReBirth RB-338 as a commercial software synthesizer.
It was the first software product to credibly emulate two Roland TB-303 Bass Line
synthesizers alongside a Roland TR-808 and TR-909 drum machine within a single application.

ReBirth ran on Windows 95/98 and Apple Macintosh (68k and PowerPC) with a modest system
requirement of 16 MB RAM and a 100 MHz processor.

## Core Features

- **Dual TB-303 emulation** — two independent 303 modules, each with its own pattern sequencer
- **TR-808 drum machine** — 11 instruments, 16-step pattern sequencer, accent and shuffle
- **TR-909 drum machine** — shared sequencer with configurable swing and flam
- **Song mode** — chain up to 16 patterns per device
- **Real-time control** — cutoff, resonance, envelope, and accent knobs fully automatable
- **MIDI sync** — slave to external MIDI clock (MIDI In required)

## File Format

Songs are stored as `.rbs` binary files containing:
- Pattern data for all devices
- Song arrangement (pattern chains)
- Knob automation data
- Tempo and swing settings

## Known Limitations

- No VST/DirectX plugin support; standalone only
- Polyphony limited to the TB-303 monophonic architecture
- No audio export; output via sound card only
- No undo history

## Trivia

The "338" designation is a nod to the combined model numbers of the emulated hardware:
**303 + 808 + 909 ≈ 338** (a playful Propellerhead shorthand).
